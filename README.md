# Data Asset Hub

An enterprise data asset management platform demo for Starbucks. Built with pure HTML/CSS/JavaScript — no build tools required.

## Pages

| Page | File | Description |
|------|------|-------------|
| **Homepage** | `index.html` | Data overview, matrix, governance policies |
| **Asset Shelf** | `shelf.html` | Browse & filter all data assets |
| **Glossary Detail** | `detail-glossary.html` | KPI/Term definition, calculation logic |
| **Dictionary Detail** | `detail-dictionary.html` | Table schema with field-level info |
| **Reference Doc Detail** | `detail-document.html` | Business guide with TOC & screenshots |
| **Data Application** | `apply-data.html` | Request data access / L3 permission |
| **Create Asset** | `apply-asset.html` | Submit new data asset for approval |
| **Admin - Asset List** | `admin-list.html` | Manage all assets (publish/archive) |
| **Admin - Edit** | `admin-edit.html` | Edit asset content by type |
| **Admin - History** | `admin-history.html` | Change log & audit trail |
| **Admin - Config** | `admin-config.html` | Configure homepage modules & links |
| **Admin - Access** | `admin-access.html` | View access records & analytics |

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/hohelka/data-asset-demo.git
   ```
2. Open `index.html` in a browser, or use a local server:
   ```bash
   npx serve .
   ```

## Color Palette

- **Starbucks Green** `#00704A` — Primary brand color
- **Dark Green** `#1E3932` — Navigation & headings
- **Gold** `#CBA258` — Accents & warnings
- **Brown** `#6B4226` — Reference document tags
