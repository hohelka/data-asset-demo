Customer segmentation is a foundational capability for personalized marketing, loyalty program optimization, and strategic planning. This guide consolidates the customer segmentation methodology used across all business units at Starbucks China.

The primary goal is to provide a unified framework for understanding customer behavior, value distribution, and lifecycle stages, enabling cross-functional teams to leverage consistent segments in reporting, campaign targeting, and product development.

AttributeValue
ScopeAll Starbucks China retail customers with registered Rewards accounts
Data SourcesCRM System, POS Transaction Data, Mobile App Events
Refresh CycleMonthly (1st business day)

Segmentation Methodology

The model uses an enhanced RFM (Recency, Frequency, Monetary) framework:

DimensionDefinitionScore Range
RecencyDays since last purchase1–5 (5 = most recent)
FrequencyTransactions in last 90 days1–5 (5 = most frequent)
MonetaryTotal spend in last 90 days1–5 (5 = highest spend)

Application Scenarios

Scenario 1: Targeted Campaign

Use Diamond/Gold segments for premium product launches and exclusive offers. Green tier for re-engagement campaigns.

Scenario 2: Loyalty Tier Review

Quarterly review of tier migration patterns. Identify at-risk Diamond customers with declining frequency.

Scenario 3: Store Planning

Analyze customer segment distribution by geography to optimize store format decisions (Reserve vs. Standard).

System Operation

  1. Log in to the Starbucks Analytics Portal → Navigate to Customer Insights module.
  2. Select Segmentation tab → choose time period and region filter.
  3. Click Generate Report to view segment distribution. Export in CSV or PDF.
  4. For API access, use endpoint /api/v2/customer/segments with your data token.

客户分群是个性化营销、会员计划优化和战略规划的基础能力。本指南汇总了星巴克中国各业务部门统一采用的客户分群方法论。

主要目标是建立一套统一框架，用于理解客户行为、价值分布和生命周期阶段，使跨职能团队能够在报表分析、营销活动定向和产品研发中使用一致的分群结果。

属性内容
适用范围所有已注册星享俱乐部会员的星巴克中国零售客户
数据来源CRM 系统、POS 交易数据、移动应用行为事件
更新周期每月（第 1 个工作日）

分群方法论

本模型采用增强版 RFM（近度、频度、货值）框架：

维度定义分值范围
近度（Recency）距最近一次消费的天数1–5（5 = 最近）
频度（Frequency）近 90 天内交易笔数1–5（5 = 最频繁）
货值（Monetary）近 90 天内消费总额1–5（5 = 消费最高）

应用场景

场景一：精准营销活动

钻石级 / 黄金级会员用于高端新品发布及专属优惠活动；绿星级用于再激活营销活动。

场景二：会员等级季度审核

季度审核等级迁移规律，识别消费频次下滑的高风险钻石会员，及时发起挽留策略。

场景三：门店选址规划

按地域分析客户分群分布，为门店类型决策（尊享店 vs 标准店）提供数据支撑。

系统操作

  1. 登录星巴克分析门户 → 导航至客户洞察模块。
  2. 选择分群标签页 → 选择时间范围与地区筛选条件。
  3. 点击生成报告查看分群分布，可导出为 CSV 或 PDF 格式。
  4. 如需 API 访问，请使用接口 /api/v2/customer/segments，并携带您的数据令牌。
